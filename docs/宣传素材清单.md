## 三、截图小技巧

1. **填好配置再截**：截图前确保 Agnes API Key 和又拍云已配置，否则生成会失败，画面不完整。
2. **用真实有质感的提示词**：文生图用具体场景（如「赛博朋克城市夜景，霓虹灯，雨天反射」），比「一只猫」更有说服力。
3. **隐藏敏感信息**：截图前确认设置页里不要露出完整 API Key（用遮挡或打码）。
4. **暗色主题统一**：应用是深色 UI，截图时保持暗色，视觉更统一高级。
5. **画布截图要「满」**：无限画布截图前多放几个节点，排布得丰满些，避免空荡。
