☁️ 又拍云 FORM API 配置指南

AURA Studio v2.0.0 · 适用版本 2.0.0+

🎯 你需要准备什么？

AURA Studio 使用又拍云 FORM API 前端直传（不消耗服务器带宽），你需要获取以下 4 个信息 填入设置：

配置项说明示例
操作员名称（Access Key）操作员账号test-operator
表单 API 密钥（Secret Key）表单上传专用密钥（不是操作员密码！）xxxxxxxxxxxxxxxx
服务名称（Bucket）云存储服务名称aura-studio-releases
加速域名（CDN）文件访问域名aura-studio-releases.test.upcdn.net

💡 接入端点（Endpoint）：仅 S3 兼容模式需要，使用 FORM API 直传时可留空。

📦 第一步：注册又拍云账号

1.1 访问注册页面

打开又拍云注册页面完成注册：

👉 https://console.upyun.com/register/?invite=R3d6kOy0E

1.2 填写注册信息

  1. 输入手机号 / 邮箱
  2. 设置密码
  3. 勾选用户协议
  4. 点击 注册

1.3 完成实名认证

⚠️ 必须完成实名认证才能使用云存储服务！
  1. 登录后，进入 [账号管理] → [实名认证]
  2. 选择 个人认证 或 企业认证
  3. 上传身份证 / 营业执照
  4. 等待审核（通常 1-2 个工作日）

🗂️ 第二步：创建云存储服务（Bucket）

2.1 进入云存储控制台

  1. 登录 又拍云控制台
  2. 点击左侧菜单 [云产品] → [云存储]
  3. 点击 [创建服务] 按钮

2.2 填写服务信息

字段怎么填说明
服务名称随便起，如 aura-studio这就是你的 Bucket 名称
应用场景选 网页图片 或 文件下载仅作为管理标记，不影响功能
存储类型选 标准类型普通存储，成本低
加速区域选 国内加速（国内用户）或 全球加速（海外用户）CDN 加速范围
授权操作员选 新建操作员（后面要用）见第三步

2.3 确认创建

点击 [创建]，几秒钟后服务就建好了！

📸 截图保存：创建完成后，在服务列表可以看到你的 服务名称（Bucket） 和 加速域名，先截图保存！

👤 第三步：创建操作员并获取表单 API 密钥

⚠️ 重要：表单 API 密钥 ≠ 操作员密码，这是两个东西！
AURA Studio 使用 FORM API 前端直传，需要的是表单 API 密钥，不是操作员登录密码。

3.1 创建操作员

方式一：创建服务时同时创建（推荐）

如果在 2.2 步骤选择了"新建操作员"，会弹出创建框：

  1. 填写 操作员名称（如 aura-uploader）
  2. 设置 操作员密码（随便设，后面不用它）
  3. 勾选权限：可读 ✅、可写 ✅、可删除 ✅
  4. 点击 确定

方式二：在操作员管理页面创建

  1. 进入 [账号] → [账号管理] → [操作员管理]
  2. 点击 [创建操作员]
  3. 填写操作员名称和密码
  4. 点击 确定

3.2 授权操作员到服务

⚠️ 操作员创建后，需要授权给具体的服务才能使用！
  1. 进入 [云存储] → 点击你的服务 → [配置] → [操作员授权]
  2. 点击 [授权操作员]
  3. 选择刚才创建的操作员
  4. 勾选权限：可读 ✅、可写 ✅
  5. 点击 确定

3.3 获取表单 API 密钥（关键步骤！）

⚠️ 表单 API 密钥在 [操作员管理] 页面获取，不是创建操作员时给的密码！
  1. 进入 [账号] → [账号管理] → [操作员管理]
  2. 点击操作员名称（进入详情页，不是点编辑）
  3. 切换到 [表单 API] 标签
  4. 点击 [创建密钥]
  5. 复制生成的 表单 API 密钥（只显示一次，务必保存！）
配置项对应又拍云字段在哪里看
Access Key (AK)操作员名称又拍云控制台 → [操作员管理] 页面
Secret Key (SK)表单 API 密钥[操作员管理] → 点击操作员 → [表单 API] 标签
💡 重要提示：表单 API 密钥创建时只会显示一次，请务必保存好！如果忘记了，只能去 [表单 API] 标签 重新创建密钥。

🌐 第四步：获取加速域名（CDN Domain）

4.1 查看默认加速域名

  1. 进入 [云存储] → 点击你的服务
  2. 在服务列表页面，可以看到 加速域名 栏位

又拍云会给你分配一个默认加速域名，格式类似：

http://your-service-name.test.upcdn.net

这就是你的 CDN 域名，填到 AURA Studio 的 加速域名 字段。

4.2 （可选）绑定自定义域名

如果你想用自己的域名（如 cdn.yourdomain.com）：

  1. 进入服务 → [配置] → [域名管理]
  2. 点击 [添加域名]
  3. 输入你的域名（需已完成 ICP 备案）
  4. 按提示完成 CNAME 解析
  5. 解析生效后，用自定义域名填入 AURA Studio

⚙️ 第五步：在 AURA Studio 中填写配置

5.1 打开配置界面

  1. 启动 AURA Studio
  2. 点击右上角 [设置] 图标（⚙️）
  3. 选择 [又拍云配置] 标签页

5.2 填写配置信息

AURA Studio 字段填写内容获取位置
操作员名称（Access Key）操作员名称又拍云控制台 → [操作员管理]
表单 API 密钥（Secret Key）表单 API 密钥[操作员管理] → 点击操作员 → [表单 API] 标签
服务名称（Bucket）服务名称又拍云控制台 → [云存储] → 服务列表
加速域名（CDN）加速域名又拍云控制台 → [云存储] → 服务列表 → 加速域名栏
接入端点可留空FORM API 模式不需要
⚠️ 常见错误：把"操作员密码"填到了"表单 API 密钥"字段！这两个是不同的，请务必到 [表单 API] 标签获取正确的密钥。

5.3 测试连接

填写完成后，点击 [测试连接] 按钮：

  • ✅ 提示"连接成功" → 配置正确，可以开始用了！
  • ❌ 提示错误 → 检查以下项：
    1. 操作员名称是否填写正确（注意首尾不要有空格）
    2. 表单 API 密钥 是否填写正确（⚠️ 不是操作员密码！）
    3. 操作员是否有 可写 权限
    4. Bucket 名称是否拼写正确
    5. 域名是否填写正确

🧪 第六步：测试图片上传

配置完成后，测试一下又拍云是否正常工作：

6.1 在主界面测试

  1. 进入 [图像生成] 标签页
  2. 点击 [上传参考图]
  3. 选择一张本地图片
  4. 如果上传成功，图片会显示在预览区，并且地址是以 http://your-bucket.test.upcdn.net/... 开头的

6.2 在画布中测试

  1. 打开 画布
  2. 添加一个 图片节点
  3. 双击节点上传图片
  4. 如果上传成功，节点会显示图片预览

🆘 常见问题 FAQ

Q1：又拍云收费吗？

答：又拍云有免费额度！

  • 存储：10 GB 免费
  • 流量：15 GB/月 免费
  • 超出后：存储 ¥0.12/GB/月，流量 ¥0.5/GB

对于小范围测试（100 人以内），免费额度完全够用！

Q2：表单 API 密钥忘记了怎么办？

答：去又拍云控制台重新创建密钥（旧密钥会失效）：

  1. 进入 [账号] → [账号管理] → [操作员管理]
  2. 点击对应的操作员
  3. 切换到 [表单 API] 标签
  4. 点击 [创建密钥]
  5. 创建后，更新 AURA Studio 中的 表单 API 密钥 配置
⚠️ 重新创建密钥后，旧密钥立即失效，请同步更新 AURA Studio 配置！

Q3：为什么上传图片失败？

答：按以下顺序排查：

  1. 检查 操作员名称 是否填写正确
  2. 检查 表单 API 密钥 是否填写正确（⚠️ 不是操作员密码！这是最常见的错误）
  3. 检查操作员是否有 可写 权限
  4. 检查 Bucket 名称是否正确
  5. 检查 域名 是否填写正确
  6. 按 F12 打开开发者工具，查看 Console 标签页的错误信息

Q4：可以用其他云存储吗？

答：可以！AURA Studio 也支持：

  • 腾讯云 COS
  • AWS S3
  • MinIO（本地部署）

配置方法类似，填写对应的 AK / SK / Bucket / Endpoint 即可。

Q5：又拍云的图片可以直接在 AURA Studio 中使用吗？

答：可以！上传到又拍云的图片会通过 CDN 加速，在 AURA Studio 中加载速度很快。

图片 URL 格式：

http://your-bucket.test.upcdn.net/ai-studio/abc123.png
